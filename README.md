# TypeScript Node Mongo CRUD

This repository contains a TypeScript Node.js application with CRUD operations using MongoDB.

## Steps to run

1. Install dependencies:
   ```
   npm install
   ```

2. Compile the code:
   ```
   npm run build
   ```

3. Start the server:
   ```
   npm run start
   ```

You can access the server at `http://localhost:8080`.
